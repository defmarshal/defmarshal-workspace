**Solution:** Migrated to system crontab:
```bash
0 */2 * * * cd /home/ubuntu/.openclaw/workspace && ./agents/meta-agent.sh --once >> memory/meta-agent.log 2>&1
```

- Removed OpenClaw cron job (`openclaw cron rm`)

**Impact:**
- Zero API cost for meta-agent orchestration
- No more "rate limit reached" messages from this source
- Child agent spawns still use OpenRouter but with `spawn_agent_safe` backoff
